中国主数据咨询、实施公司列表 ​

本文档系统整理了 中国市场常见的主数据管理（MDM, Master Data Management）咨询、实施与平台提供商，覆盖从 战略规划、治理设计、平台建设到实施交付 的完整能力版图。

适用对象：

  • 甲方企业进行主数据项目选型、RFP 编制
  • 咨询/方案人员进行市场与厂商能力梳理
  • 数据治理、数据中台、集团数字化相关项目

一、厂商分类说明 ​

为便于理解与选型，本文将厂商按 主数据项目中的核心角色 分为三类：

  • A 类：咨询 + 战略规划型 以主数据战略、治理体系、组织与制度设计为核心，偏前期规划。

  • B 类：本地实施 & 平台交付型 以主数据系统建设、实施交付、平台落地为核心，最常作为项目主承包方。

  • C 类：平台提供商 + 数据治理体系支撑 提供成熟主数据/数据治理平台，通常结合生态伙伴实施。

实际项目中，常见组合方式为：A 类咨询 + B 类实施 + C 类平台。

A) 咨询 + 战略规划 ​

定位：主数据顶层设计、治理体系规划、标准与制度设计 适合阶段：主数据建设 0→1、集团治理体系搭建

1. CCID Consulting（赛迪顾问） ​

  • 官网：https://www.ccidconsulting.com

  • 公司定位：国家级背景的综合咨询机构

  • 主数据相关能力：

    • 数据治理与数据标准体系设计
    • 主数据战略规划、路线图制定
    • 行业数字化与信息化顶层设计

  • 典型场景：

    • 国企/央企主数据治理规划
    • 集团级数据治理体系设计

2. 软通动力（iSoftStone） ​

  • 官网：https://www.isoftstone.com

  • 公司定位：大型本土 IT 咨询与技术服务商

  • 主数据相关能力：

    • 数据治理咨询
    • 主数据规范与流程设计
    • 与数据中台、系统集成结合实施

  • 典型场景：

    • 大型企业数字化转型
    • 数据治理 + IT 系统整合项目

B) 本地实施 & 平台交付 ​

定位：主数据项目的核心交付方 特点：既懂业务，又能落系统，国内主数据项目的主力厂商

1. 汉得（HAND） ​

  • 官网：https://www.hand-china.com

  • 公司定位：国内领先的企业 IT 咨询与实施服务商

  • 主数据能力：

    • SAP MDG / S4 主数据治理
    • Oracle / 多 ERP 主数据整合
    • 自研主数据与数据平台

  • 优势领域：

    • 制造业、能源、化工
    • 大型集团、SAP 体系客户

  • 关键词：SAP 主数据、集团主数据治理

2. 英诺森（Inossem / InData） ​

  • 官网：https://www.inossem.com

  • 公司定位：工业与制造领域的数据治理与主数据专家

  • 主数据能力：

    • 物料主数据、设备主数据治理
    • BOM / 项目型主数据
    • 主数据清洗、标准化与质量治理

  • 优势领域：

    • 制造业、能源、工程建设

  • 关键词：工业主数据、物料与设备主数据

3. 盟拓数字科技（Movit Tech） ​

  • 官网：https://www.movit-tech.com

  • 公司定位：企业数字化与数据治理解决方案提供商

  • 主数据能力：

    • 主数据平台建设
    • 数据治理与系统集成

  • 优势领域：

    • 制造与集团型企业

4. 用友（Yonyou） ​

  • 官网：https://www.yonyou.com

  • 公司定位：国内大型企业管理软件与平台厂商

  • 主数据能力：

    • 集团级主数据中枢建设
    • 多组织、多法人主数据治理
    • 与财务、人力、供应链深度融合

  • 优势领域：

    • 国企/央企
    • 大型集团企业

  • 关键词：集团管控、主数据中台

5. 金蝶（Kingdee） ​

  • 官网：https://www.kingdee.com

  • 公司定位：企业管理软件与云服务厂商

  • 主数据能力：

    • 主数据管理平台产品
    • 与金蝶 ERP 深度集成

  • 优势领域：

    • 使用金蝶 ERP 的企业
    • 政企与大型企事业单位

  • 关键词：主数据平台、ERP 一体化

C) 平台提供商 + 数据治理体系支撑 ​

定位：提供成熟平台与治理方法论，实施多依赖生态伙伴

1. 华为 / 华为云 ​

  • 官网：https://www.huawei.com

  • 华为云：https://www.huaweicloud.com

  • 公司定位：数字化与云平台厂商

  • 主数据能力：

    • 数据治理与主数据管理平台
    • 数据标准、质量、资产统一治理

  • 特点：

    • 强方法论、强平台
    • 多采用“平台 + 生态伙伴”交付模式

  • 典型场景：

    • 云原生数据治理
    • 大型数据平台建设

2. 普元（Primeton MDM） ​

  • 官网：https://www.primeton.com

  • 公司定位：国产中间件与数据治理平台厂商

  • 主数据能力：

    • 国产主数据管理平台
    • 数据治理与集成能力

  • 典型场景：

    • 国产化替代
    • 政企与国企客户

四、选型建议（简要） ​

  • 主数据顶层设计 / 咨询规划：A 类厂商
  • 主数据系统实施与交付：B 类厂商
  • 统一数据治理平台建设：C 类厂商

实际项目中，通常采用 “咨询 + 实施 + 平台” 组合模式，以降低项目风险。

五、备注 ​

  • 本清单聚焦 中国市场常见厂商，不包含纯国际厂商（如 SAP、IBM、Informatica）。
  • 厂商能力会随产品与组织调整，建议在正式选型前进行 RFP 与 PoC 验证。
