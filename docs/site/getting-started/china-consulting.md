# List of MDM Consulting and Implementation Companies in China

This document systematically organizes **common Master Data Management (MDM) consulting, implementation, and platform providers in the Chinese market**, covering the complete capability map from **strategic planning, governance design, platform construction to implementation delivery**.

> Applicable Audience:
>
> * Enterprise customers conducting MDM selection and RFP preparation
> * Consulting/Solution personnel for market and vendor capability landscaping
> * Data governance, data middle platform, and group digitalization related projects

## I. Vendor Classification

To facilitate understanding and selection, this document divides vendors into three categories based on **core roles in MDM projects**:

* **Type A: Consulting + Strategic Planning**
  Focuses on MDM strategy, governance system, organization, and system design. Geared towards early-stage planning.

* **Type B: Local Implementation & Platform Delivery**
  Focuses on MDM system construction, implementation delivery, and platform landing. Most commonly serves as the main project contractor.

* **Type C: Platform Provider + Data Governance System Support**
  Provides mature MDM/Data Governance platform products, usually implemented in combination with ecosystem partners.

> In actual projects, a common combination is: **Type A Consulting + Type B Implementation + Type C Platform**.

## IV. Selection Suggestions (Brief)

* **Master Data Top-level Design / Consulting Planning**: Type A Vendors
* **Master Data System Implementation and Delivery**: Type B Vendors
* **Unified Data Governance Platform Construction**: Type C Vendors

> In actual projects, the **"Consulting + Implementation + Platform" combination mode** is usually adopted to reduce project risk.

## V. Remarks

* This list focuses on **common vendors in the Chinese market** and does not include purely international vendors (such as SAP, IBM, Informatica).
* Vendor capabilities may change with product and organization adjustments. It is recommended to conduct RFP and PoC verification before formal selection.
