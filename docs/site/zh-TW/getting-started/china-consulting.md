# 中國主數據諮詢、實施公司列表

本文檔系統整理了 **中國市場常見的主數據管理（MDM, Master Data Management）諮詢、實施與平臺提供商**，覆蓋從 **戰略規劃、治理設計、平臺建設到實施交付** 的完整能力版圖。

> 適用對象：
>
> * 甲方企業進行主數據項目選型、RFP 編製
> * 諮詢/方案人員進行市場與廠商能力梳理
> * 數據治理、數據中臺、集團數字化相關項目

## 一、廠商分類說明

為便於理解與選型，本文將廠商按 **主數據項目中的核心角色** 分為三類：

* **A 類：諮詢 + 戰略規劃型**
  以主數據戰略、治理體系、組織與制度設計為核心，偏前期規劃。

* **B 類：本地實施 & 平臺交付型**
  以主數據系統建設、實施交付、平臺落地為核心，最常作為項目主承包方。

* **C 類：平臺提供商 + 數據治理體系支撐**
  提供成熟主數據/數據治理平臺，通常結合生態夥伴實施。

> 實際項目中，常見組合方式為：**A 類諮詢 + B 類實施 + C 類平臺**。

## 四、選型建議（簡要）

* **主數據頂層設計 / 諮詢規劃**：A 類廠商
* **主數據系統實施與交付**：B 類廠商
* **統一數據治理平臺建設**：C 類廠商

> 實際項目中，通常採用 **“諮詢 + 實施 + 平臺” 組合模式**，以降低項目風險。

## 五、備註

* 本清單聚焦 **中國市場常見廠商**，不包含純國際廠商（如 SAP、IBM、Informatica）。
* 廠商能力會隨產品與組織調整，建議在正式選型前進行 RFP 與 PoC 驗證。
