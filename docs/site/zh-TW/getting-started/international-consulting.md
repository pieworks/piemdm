# 國際主數據（MDM）諮詢、實施公司清單

本文檔整理 **國際市場常用的主數據管理（MDM, Master Data Management）相關廠商**，涵蓋 **諮詢公司、平臺廠商、實施服務商**。清單以國際主流選型視角為主，同時 **包含在國際項目中常見的中國公司**。

> 適用對象：
>
> * 跨國企業 / 出海企業主數據選型
> * 全球主數據治理（Global MDM）項目
> * 對標國際廠商能力、做方案或競品分析

---

## 一、廠商分類說明

國際主數據項目通常按以下角色組合推進：

* **A 類：國際諮詢公司（戰略 + 治理）**
  提供全球主數據戰略、治理模型、組織與流程設計。

* **B 類：國際 MDM 平臺廠商**
  提供成熟、國際化的主數據管理軟件產品，是項目技術核心。

* **C 類：國際實施服務商 / SI**
  負責平臺實施、系統集成、本地化交付。

> 常見組合：**A 類諮詢 + B 類平臺 + C 類實施（或同一廠商多角色）**。

## A) 國際諮詢公司（戰略 + 治理）

**定位**：全球主數據戰略、數據治理方法論、跨區域治理模型設計

* **官網**：[https://www.accenture.com](https://www.accenture.com)
* **公司定位**：全球最大數字化與管理諮詢公司之一
* **主數據能力**：
  * Global MDM 戰略與治理模型

  * MDM 與 ERP / Data Platform 深度結合
* **典型場景**：
