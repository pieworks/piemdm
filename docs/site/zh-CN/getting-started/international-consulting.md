# 国际主数据（MDM）咨询、实施公司清单

本文档整理 **国际市场常用的主数据管理（MDM, Master Data Management）相关厂商**，涵盖 **咨询公司、平台厂商、实施服务商**。清单以国际主流选型视角为主，同时 **包含在国际项目中常见的中国公司**。

> 适用对象：
>
> * 跨国企业 / 出海企业主数据选型
> * 全球主数据治理（Global MDM）项目
> * 对标国际厂商能力、做方案或竞品分析

---

## 一、厂商分类说明

国际主数据项目通常按以下角色组合推进：

* **A 类：国际咨询公司（战略 + 治理）**
  提供全球主数据战略、治理模型、组织与流程设计。

* **B 类：国际 MDM 平台厂商**
  提供成熟、国际化的主数据管理软件产品，是项目技术核心。

* **C 类：国际实施服务商 / SI**
  负责平台实施、系统集成、本地化交付。

> 常见组合：**A 类咨询 + B 类平台 + C 类实施（或同一厂商多角色）**。
