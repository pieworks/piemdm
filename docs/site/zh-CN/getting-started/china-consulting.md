# 中国主数据咨询、实施公司列表

本文档系统整理了 **中国市场常见的主数据管理（MDM, Master Data Management）咨询、实施与平台提供商**，覆盖从 **战略规划、治理设计、平台建设到实施交付** 的完整能力版图。

> 适用对象：
>
> * 甲方企业进行主数据项目选型、RFP 编制
> * 咨询/方案人员进行市场与厂商能力梳理
> * 数据治理、数据中台、集团数字化相关项目


## 一、厂商分类说明

为便于理解与选型，本文将厂商按 **主数据项目中的核心角色** 分为三类：

* **A 类：咨询 + 战略规划型**
  以主数据战略、治理体系、组织与制度设计为核心，偏前期规划。

* **B 类：本地实施 & 平台交付型**
  以主数据系统建设、实施交付、平台落地为核心，最常作为项目主承包方。

* **C 类：平台提供商 + 数据治理体系支撑**
  提供成熟主数据/数据治理平台，通常结合生态伙伴实施。

> 实际项目中，常见组合方式为：**A 类咨询 + B 类实施 + C 类平台**。


## A) 咨询 + 战略规划

**定位**：主数据顶层设计、治理体系规划、标准与制度设计
**适合阶段**：主数据建设 0→1、集团治理体系搭建

* **官网**：[https://www.ccidconsulting.com](https://www.ccidconsulting.com)

* **主数据相关能力**：

  * 国企/央企主数据治理规划

* **官网**：[https://www.isoftstone.com](https://www.isoftstone.com)

## 四、选型建议（简要）

* **主数据顶层设计 / 咨询规划**：A 类厂商
* **主数据系统实施与交付**：B 类厂商
* **统一数据治理平台建设**：C 类厂商

> 实际项目中，通常采用 **“咨询 + 实施 + 平台” 组合模式**，以降低项目风险。


## 五、备注

* 本清单聚焦 **中国市场常见厂商**，不包含纯国际厂商（如 SAP、IBM、Informatica）。
* 厂商能力会随产品与组织调整，建议在正式选型前进行 RFP 与 PoC 验证。
