# 主数据管理系统（MDM）系统清单

> 本文档整理 **主数据管理系统（MDM, Master Data Management）** 的主流产品，覆盖：
>
> * 🌍 国际厂商 / 🇨🇳 中国厂商
> * 💰 商业收费 / 🔓 开源（含社区版）
> * 🧩 多域 MDM / 客户、产品、供应商等专用 MDM

适用于 **主数据平台选型、技术对标、方案设计、RFP 编制** 等场景。


## 一、国际主流商业 MDM 系统（收费）

### 1. Informatica MDM

* **官网**：[https://www.informatica.com](https://www.informatica.com)
* **类型**：多域 MDM（Multi-domain MDM）
* **特点**：

  * 全球市场占有率高
  * 数据治理、质量、集成能力完整
* **适用场景**：大型跨国企业、集团型 MDM


### 2. SAP Master Data Governance（MDG）

* **官网**：[https://www.sap.com](https://www.sap.com)
* **类型**：ERP 内嵌型 MDM
* **特点**：

  * 与 SAP S/4HANA 深度绑定
  * 制造业、集团客户广泛


### 3. Oracle Master Data Management

* **官网**：[https://www.oracle.com](https://www.oracle.com)
* **类型**：企业级多域 MDM
* **特点**：

  * Oracle 应用与数据库生态强


### 4. Stibo Systems STEP

* **官网**：[https://www.stibosystems.com](https://www.stibosystems.com)
* **类型**：PIM + MDM
* **特点**：

  * 零售、消费品、电商强项


### 5. Reltio Connected Data Platform

* **官网**：[https://www.reltio.com](https://www.reltio.com)
* **类型**：云原生 MDM / Customer 360
* **特点**：

  * 客户主数据、关系型数据建模


### 6. Profisee MDM

* **官网**：[https://www.profisee.com](https://www.profisee.com)
* **类型**：多域 MDM
* **特点**：

  * Azure / Microsoft 生态友好


### 7. Semarchy xDM

* **官网**：[https://www.semarchy.com](https://www.semarchy.com)
* **类型**：模型驱动 MDM
* **特点**：

  * 轻量、快速实施


### 8. Talend MDM

* **官网**：[https://www.talend.com](https://www.talend.com)
* **类型**：MDM + 数据集成


### 9. IBM InfoSphere MDM

* **官网**：[https://www.ibm.com](https://www.ibm.com)
* **类型**：企业级 MDM


## 二、中国商业主数据管理系统

### 1. 用友 主数据中台 / MDM

* **官网**：[https://www.yonyou.com](https://www.yonyou.com)
* **类型**：集团级 MDM
* **特点**：

  * 多组织、多法人治理
  * 与财务、供应链深度集成


### 2. 金蝶 主数据管理平台

* **官网**：[https://www.kingdee.com](https://www.kingdee.com)
* **类型**：ERP 绑定型 MDM


### 3. 普元 MDM（Primeton）

* **官网**：[https://www.primeton.com](https://www.primeton.com)
* **类型**：国产多域 MDM
* **特点**：

  * 国产化替代
  * 政企客户


### 4. 英诺森 InData MDM

* **官网**：[https://www.inossem.com](https://www.inossem.com)
* **类型**：工业主数据 MDM
* **特点**：

  * 物料 / 设备 / BOM 主数据


### 5. 汉得 主数据平台（HAND MDM）

* **官网**：[https://www.hand-china.com](https://www.hand-china.com)
* **类型**：实施型 MDM 平台


### 6. 三维天地 主数据管理系统

* **官网**：[https://www.sunwayworld.com](https://www.sunwayworld.com)
* **类型**：国产 MDM


## 三、开源 / 社区版主数据系统

> ⚠️ 多数开源 MDM 偏产品数据（PIM）或轻量 MDM，企业级使用需二次开发。

### 1. Pimcore

* **官网**：[https://pimcore.com](https://pimcore.com)
* **授权**：GPL（社区版）+ 企业版收费
* **类型**：PIM / MDM / DAM


### 2. AtroCore

* **官网**：[https://www.atrocore.com](https://www.atrocore.com)
* **授权**：开源 + SaaS
* **类型**：PIM + MDM


### 3. OpenMDM

* **官网**：[https://github.com/openmdm](https://github.com/openmdm)
* **授权**：开源
* **类型**：基础 MDM 框架


### 4. HiveMDM

* **授权**：开源
* **类型**：技术型 MDM 方案


### 5. PieMDM

* **官网 / 项目地址**：[https://github.com/pieteams/piemdm](https://github.com/pieteams/piemdm)
* **项目网站**：[https://pieteams.github.io/piemdm/](https://pieteams.github.io/piemdm/)
* **授权**：MIT 开源许可
* **类型**：开源企业级 MDM 系统
* **技术栈**：Go 后端 + Vue 前端
* **主要功能**：

  * 主数据建模与数据管理
  * 数据治理与访问控制
  * 工作流 / 审批管理
  * 系统集成 & API 支持
* **适用场景**：

  * 适合尝试性、PoC 或希望基于源码扩展的主数据项目
  * 社区实践、自主研发型企业


## 四、按维度快速索引

### 商业 vs 开源

* **商业**：Informatica / SAP / Oracle / Stibo / 用友 / 普元
* **开源**：Pimcore / AtroCore / OpenMDM / PieMDM

### 国际 vs 中国

* **国际**：Informatica / SAP / Oracle / Stibo / Reltio
* **中国**：用友 / 金蝶 / 普元 / 英诺森 / 汉得 / PieMDM


## 五、选型建议（简要）

* **全球集团 / 复杂治理**：Informatica / SAP MDG
* **制造业主数据**：SAP MDG / 英诺森
* **国产化要求**：用友 / 普元 / 金蝶 / PieMDM
* **低成本探索 / 原型验证**：Pimcore / AtroCore / PieMDM
